1️⃣ index.html — მთავარი

დიდი გმირის ბლოკი აკადემიის ფოტოთი, სლოგანით ან მისიით.
ღილაკები/ბმულები კურსებსა და პროგრამებზე წვდომისთვის.
მოკლე ბლოკი "რატომ უნდა აგვირჩიოთ ჩვენ" (ბარათებში 3-4 შეღავათი).


2️⃣ programs.html — პროგრამები / კურსები

კურსების სია ბარათებში (CSS Grid ან Flexbox).
თითოეული ბარათი შეიცავს: სურათს, კურსის სათაურს და მოკლე აღწერას.
ღილაკი "შეიტყვეთ მეტი" (შეიძლება იმავე გვერდზე გადახვიდეთ, მხოლოდ საჩვენებლად).


3️⃣ teachers.html — მასწავლებლები

მასწავლებლების ფოტოები ბადეში.
ფოტოს ქვეშ: სახელი, საგანი/სპეციალობა, მოკლე ბიოგრაფია.
მაუსის გადატანა ფოტოს მცირედი გადიდებით ან ჩრდილით.


4️⃣ student-life.html — სტუდენტური ცხოვრება

ღონისძიებების ფოტო გალერეა (CSS Grid).
შეგიძლიათ დაამატოთ მარტივი CSS სლაიდერი (@keyframes animation).


5️⃣ about.html — ჩვენს შესახებ

აკადემიის ისტორია, დანიშნულება და ღირებულებები.
შეგიძლიათ შექმნათ CSS დროის სქემი (ვერტიკალური ან ჰორიზონტალური დროის სქემი თარიღებითა და მოვლენებით).
შენობის ფოტო, ლოგო.


6️⃣ contact.html — კონტაქტები

მისამართი, რუკა (რუკის სურათი), ტელეფონის ნომრები, ელ. ფოსტა.
საკონტაქტო ფორმა (მხოლოდ მარკირება, JavaScript-ის გარეშე).
ფორმის ველის სტილი: :focus effects.
